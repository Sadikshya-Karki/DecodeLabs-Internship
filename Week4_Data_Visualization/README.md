# Week 4: Data Visualization

Performed data visualization on the e-commerce orders dataset (1,200 records) as part of my Data Analytics internship at DecodeLabs.

## Goal
Create clear and meaningful visual representations of the dataset to communicate key business insights through storytelling with data, using appropriate chart types for each insight.

## What was done
- Created a pie chart to show order status distribution and highlight the low delivery rate
- Built a horizontal bar chart comparing revenue across referral sources
- Plotted a line chart tracking monthly revenue trends from January 2023 to June 2025
- Created a stacked bar chart to show order status breakdown across all products
- Built a box plot to compare TotalPrice spread and outliers across product categories
- Created a scatter plot to explore the relationship between UnitPrice and TotalPrice
- Plotted an area chart to visualize revenue accumulation over time
- Built a bar chart comparing revenue generated by each payment method
- Created a donut chart to show coupon code usage distribution
- Built a grouped bar chart comparing cancelled vs delivered orders per product

## Key findings
- Only 19.3% of orders are successfully delivered, while cancelled and returned orders together account for 41.4% of all transactions
- Instagram generates the highest revenue at 275,285, confirming it is the most valuable marketing channel
- Monthly revenue shows no strong seasonal pattern, fluctuating consistently across all 30 months
- Credit Card drives the most revenue at 263,848, with the highest average order value at 1127.55
- FREESHIP is the most used coupon at 34.8% of coupon orders, showing that free shipping is a strong purchase motivator
- Cancellations exceed deliveries for almost every product, with Chair being the worst at 45 cancellations vs 33 deliveries

## Key learning
Choosing the right chart type matters as much as the data itself. A pie chart communicates proportion instantly, while a box plot reveals spread and outliers that a bar chart would completely hide. The same dataset tells a very different story depending on how it is visualized, and the goal is always to make the insight obvious to someone seeing the data for the first time.
