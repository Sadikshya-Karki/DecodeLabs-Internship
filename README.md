# DecodeLabs Data Analytics Internship

Data analytics project completed during my Industrial Training internship at DecodeLabs (Batch 2026). The project covers the full analytics workflow on a single e-commerce orders dataset, from cleaning raw data to SQL analysis to final visualizations.

## About the Dataset

A raw e-commerce orders dataset with 1,200 records, including fields like OrderID, Product, TotalPrice, UnitPrice, PaymentMethod, OrderStatus, ReferralSource, CouponCode, and Date.

## Objective

Turn a messy raw dataset into a clean, reliable source of truth, then explore it, query it, and visualize it to answer real business questions about order fulfillment, revenue, and customer behavior.

## Workflow

1. **Data Cleaning** – Fixed missing values, checked for duplicates, standardized date formats, and corrected numeric precision issues.
2. **Exploratory Data Analysis (EDA)** – Used descriptive statistics, distributions, outlier detection, and correlation analysis to find patterns in the cleaned data.
3. **SQL Analysis** – Wrote 18 SQL queries in MySQL to filter, group, and aggregate the data and answer specific business questions.
4. **Data Visualization** – Built 10 different charts in Python to communicate the findings clearly through storytelling with data.
